Projects

Weather App

This is a weather app built using HTML, SASS and JS. Utilizing the OpenWeatherMap API to fetch real-time data as well as 3 day forecasts for the region specified by the user. Learned a lot about how APIs and HTTP requests are returned and how to handle the various forms of response from the server.

To-Do List

This was a fun project to make. I created a To-Do list that allows sorting and searching of tasks. You can sort the tasks by "Work" or "Personal" categories. Each task is time stamped upon creation using a constructor class. Each item animates upon deletion to make clear which task is being removed.

16-Pad Drum Machine

This was by far my favorite project to make. It was originally a spin-off of a Wes Bos project for a simple sample trigger. But, I wanted to take it a bit further and add more pads. This was built using CSS Grid for layout and using the javascript play() function to trigger the samples. I'd like to expand it to allow for custom drum kits and record and playback.
